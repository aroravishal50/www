RESONAI SEARCH FOR UNITY

Resonai Visual Search is now available to Unity developers!
Resonai Visual Search is a search engine that is powered by machine understanding of geometry and structure.
It allows you to discover the content you need, quickly and intuitively.

Features:

• Visual search for 3D assets

• Easy access to a vast dataset of content

• Integration to Unity’s Asset store

• Preview of assets in your project
before you download them

• Advanced search filters such as
polycount

• Runs on Unity Editor and coming soon -
support for Unity’s Editor VR

• Free
